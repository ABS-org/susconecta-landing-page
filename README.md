# SUSConecta Landing Page

Makefile commands
```bash
network-run         #Start Networking
network-stop        #Stop Networking

webapp-image-build  #Create Webapp Docker Image
webapp-run          #Start Webapp server
webapp-logs         #Show Webapp Logs
webapp-stop         #Stop Webapp Server
webapp-bower-update #Update Webapp bower dependencies
webapp-npm-install  #Install Webapp Npm Packages
webapp-dist         #Create Webapp Dist Folder
webapp-console      #Console Webapp Container

server-run          #Start API Server
server-stop         #Stop API Server
server-npm-install  #Install API NPM Packages
server-console      #Console API Container
server-debug        #Start API with Console

monitor             #Show Conatiners Status
```
